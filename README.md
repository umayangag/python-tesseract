# Python-Tesseract
A python wrapper and server implementation for Tesseract OCR

## Requirements 
Python 3.8

## Installation 
Run the following command to install the dependencies.

#### `./install.sh`

## Run Server 
Run the following command to install the dependencies.

#### `./run_server.sh`

# Usage

POST $IP:8080/extract

    Request  : get parameter 'url' for image url
    
    Response : [text]
